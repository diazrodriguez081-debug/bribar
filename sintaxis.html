💻 Sintaxis de Java

La sintaxis son las reglas que debemos seguir para escribir correctamente un programa en Java.

📦 Crear una clase

Una clase es la estructura principal de un programa en Java.

public class MiPrograma {
}

▶️ Método main

El método main es el punto donde comienza la ejecución de un programa.

public static void main(String[] args) {
}

📝 Declarar una variable

Para crear una variable se escribe el tipo de dato, el nombre y su valor.

int edad = 16;

🖥️ Mostrar información

Se utiliza este comando para mostrar información en la pantalla.

System.out.println("Hola Java");

🔀 Condicional if

Permite ejecutar una acción cuando se cumple una condición.

if (edad >= 18) {
  System.out.println("Mayor de edad");
}

🔁 Bucle for

Permite repetir una instrucción varias veces.

for (int i = 0; i < 5; i++) {
  System.out.println(i);
}
