Ejemplo 1: Hola Mundo

Este es uno de los primeros programas que se aprende en Java. 

public class HolaMundo {

    public static void main(String[] args) {

        System.out.println("Hola Mundo");

    }

}

Ejemplo 2: Suma de dos números

En este ejemplo se utilizan variables para realizar una suma. 

public class Suma {

    public static void main(String[] args) {

        int numero1 = 10;
        int numero2 = 5;

        int resultado = numero1 + numero2;

        System.out.println("La suma es: " + resultado);

    }

}

Ejemplo 3: Condicional IF

El condicional permite tomar decisiones dentro del programa. 

public class Edad {

    public static void main(String[] args) {

        int edad = 18;

        if (edad >= 18) {

            System.out.println("Eres mayor de edad");

        } else {

            System.out.println("Eres menor de edad");

        }

    }

}

Ejemplo 4: Ciclo FOR

El ciclo FOR permite repetir una instrucción varias veces. 

public class Contador {

    public static void main(String[] args) {

        for (int i = 1; i <= 5; i++) {

            System.out.println("Número: " + i);

        }

    }

}

Ejemplo 5: Variables

Las variables permiten almacenar información dentro de un programa. 

public class Datos {

    public static void main(String[] args) {

        String nombre = "Xioma";

        int edad = 16;

        System.out.println("Mi nombre es: " + nombre);

        System.out.println("Tengo " + edad + " años");

    }

}
