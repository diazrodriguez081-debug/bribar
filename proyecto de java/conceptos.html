📘 Conceptos básicos de Java

Antes de comenzar a programar en Java, es importante conocer algunos conceptos fundamentales del lenguaje.

🔤 Variable

Una variable es un espacio en la memoria que permite almacenar información.

int edad = 16;

📦 Tipo de dato

Los tipos de datos indican qué tipo de información puede almacenar una variable.

int numero = 10;
String nombre = "Ana";

🧱 Clase

Una clase es una estructura que sirve como modelo para crear objetos.

class Persona { }

🎯 Objeto

Un objeto es una instancia creada a partir de una clase.

Persona persona1 = new Persona();

⚙️ Método

Un método es un bloque de código que realiza una tarea específica.

void saludar() { }

📋 Instrucción

Una instrucción es una orden que el programa debe ejecutar.

System.out.println( "Hola Java" );
